# Weather App

A modern weather application built using HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Real-time temperature
- Weather conditions
- Humidity and wind speed
- Dynamic weather icons
- Responsive modern UI

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
